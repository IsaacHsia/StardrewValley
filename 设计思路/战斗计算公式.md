## 核心公式

```
最终伤害 = (基础伤害 + 力量) × 倍率修正 × 受击修正 → 取整后 - 格挡 → 剩余扣血
```

## 详细结算流程（按顺序）

### 第一步：计算基础攻击值

```
攻击值 = 卡牌基础伤害 + 力量(Strength)
```

- 力量可为负数（被削弱时）
- 攻击值最低为 **0**，不会变成负数

### 第二步：应用攻击方倍率修正

按 **乘法叠加** 依次计算：

|效果|倍率|说明|
|---|---|---|
|**虚弱（Weak）**|×0.75|攻击方被施加虚弱时|
|**双重打击类效果**|×2|如遗物"笔尖"等|
|**其他攻击加成遗物**|各自倍率|如纸鹤扇等|

```
修正后攻击值 = 攻击值 × 0.75（若虚弱） × 其他倍率
```

> **关键**：每次乘法后 **向下取整（floor）**

### 第三步：应用受击方倍率修正

|效果|倍率|说明|
|---|---|---|
|**易伤（Vulnerable）**|×1.5|受击方处于易伤状态|
|遗物"奇异孢子"等|各自倍率|特殊遗物修正|
|遗物"金牌纸扇"（玩家防御时）|各自效果|—|

```
最终伤害值 = 修正后攻击值 × 1.5（若易伤） × 其他倍率
```

> 同样每步 **向下取整**

### 第四步：格挡结算

```
实际扣血 = max(0, 最终伤害值 - 当前格挡)
剩余格挡 = max(0, 当前格挡 - 最终伤害值)
```

- 格挡先被消耗，剩余伤害才扣血
- 格挡不会被打成负数

### 第五步：生命值变化

```
当前HP = 当前HP - 实际扣血
```

## 完整公式（一行总结）

```
实际扣血 = max(0, floor(floor((基础伤害 + 力量) × 虚弱倍率) × 易伤倍率) - 格挡)
```

## 计算示例

> 打出一张 **6 伤害** 的牌，拥有 **3 力量**，自身 **被虚弱**，对方 **有易伤**，对方 **5 格挡**

```
① 基础攻击值 = 6 + 3 = 9
② 虚弱修正   = floor(9 × 0.75) = floor(6.75) = 6
③ 易伤修正   = floor(6 × 1.5) = floor(9.0) = 9
④ 格挡结算   = 9 - 5 = 4
⑤ 实际扣血   = 4
```

## 多段攻击

多段攻击（如旋风斩打 N 次）：**每一击独立结算完整公式**，每一击独立与格挡交互。

```
第1击：max(0, 单次伤害 - 当前格挡) → 扣血，更新格挡
第2击：max(0, 单次伤害 - 剩余格挡) → 扣血，更新格挡
...以此类推
```

这就是为什么多段攻击打有格挡的敌人比单次大伤害更亏（格挡可以挡住每一击），但打无格挡敌人时效果相同。

## 特殊机制补充

| 机制              | 说明                           |
| --------------- | ---------------------------- |
| **荆棘（Thorns）**  | 受击时反弹固定伤害，不受力量/易伤影响          |
| **毒（Poison）**   | 回合开始扣血，**无视格挡**，不受力量/虚弱/易伤影响 |
| **固定伤害（如失去HP）** | 直接扣血，**无视格挡**，不受任何修正         |
| **取整规则**        | 所有中间步骤 **向下取整（floor）**       |
