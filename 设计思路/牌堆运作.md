# 基础逻辑

|操作|逻辑|
|---|---|
|**抽牌**|每回合默认从抽牌堆顶抽 5 张|
|**弃牌**|回合结束时，手牌中非 Retain 的卡全部移入弃牌堆|
|**洗牌**|当抽牌堆为空且需要抽牌时，弃牌堆**随机打乱**后成为新的抽牌堆|
|**消耗**|被消耗的卡移入消耗堆，**本场战斗不再出现**|
|**Innate**|战斗开始时，所有 Innate 卡优先进入起手手牌|
|**Ethereal**|回合结束时检查：若在手中 → 消耗（移入消耗堆）|
|**Sly**|当从手牌弃置时拦截：不进弃牌堆 → 立即免费打出|

# 状态卡与诅咒卡的污染

|卡类型|来源|清除方式|
|---|---|---|
|**状态卡** (Burn, Wound, Slimed 等)|敌人塞入|战斗结束自动移除；战斗中可通过消耗清除|
|**诅咒卡**|事件/敌人/遗物|战斗结束后仍保留；需商店花钱移除或特殊手段消耗|
